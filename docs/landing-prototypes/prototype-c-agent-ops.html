How the system works

Agent work becomes an operations problem.

Once agents can code, the hard question changes. Which task should run? Where should it run? What if it blocks another task? How do we know it tested, reviewed, and merged safely? Sandman answers those questions from the CLI.

The developer-facing model

Developers queue clear work, choose the level of isolation, watch a local portal if they want, and review durable outputs later. Sandman handles the repetitive orchestration while keeping the repository's workflow visible.

Compared with spec-driven development

Specs decide what should exist. Sandman makes agents produce it safely.

Spec Kit and Kiro popularize a critical idea: intent must be structured before code is written. Sandman complements that movement. It accepts structured intent, then focuses on isolated execution, review iteration, and merge discipline.

Before Sandman

After Sandman

Review branches and pull requests with green CI, delegated feedback, and an explicit merge gate.

Competitive landscape

Where Sandman fits among coding-agent products.

The market is converging on background agents, specs, and review automation. Sandman's differentiator is narrow and practical: developer-owned orchestration around the work, not another place to ask an AI to code.

Differentiators

The five reasons Sandman should exist.

Isolation

Every agent run belongs in a worktree or container, not in the developer's active checkout.

Observability

Status, history, logs, portal views, and event-sourced run state make AFK work inspectable.

Workflow rails

Skills encode planning, TDD, self-review, back-merge, PR review, and merge behavior.

Batch throughput

Parallel runs, dependency-aware issue selection, retries, and continue mode turn a backlog into a queue.

Review discipline

Sandman treats PR review and CI gates as first-class, not as optional cleanup after code generation.

Developer ownership

The repository, prompt files, skills, branches, and decisions remain visible to the engineering team.
