# Sandman

## Quick start

Use one mode explicitly:

```text
sandman implement
sandman tdd
sandman review
sandman delegate-review
sandman merge
sandman pr-merge
sandman continuation
```

## Modes

- `implement` -> `sandman-implement`
- `tdd` -> `sandman-tdd`
- `review` -> `sandman-review`
- `delegate-review` -> `sandman-delegate-review`
- `merge` -> `sandman-merge`
- `pr-merge` -> `sandman-pr-merge`
- `continuation` -> `sandman-continuation`

## Use

Load the matching subskill for the requested mode and follow it end to end.
