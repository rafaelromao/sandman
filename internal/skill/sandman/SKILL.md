# Sandman

## Quick start

Use one mode explicitly:

```text
sandman plan
sandman implement
sandman tdd
sandman self-review
sandman pr-review
sandman back-merge
sandman pr-merge
```

## Modes

- `plan` -> `sandman-plan`
- `implement` -> `sandman-implement`
- `tdd` -> `sandman-tdd`
- `self-review` -> `sandman-self-review`
- `pr-review` -> `sandman-pr-review`
- `back-merge` -> `sandman-back-merge`
- `pr-merge` -> `sandman-pr-merge`
- `codeindex` -> `sandman-codeindex`

## Use

Load the matching subskill for the requested mode and follow it end to end.
