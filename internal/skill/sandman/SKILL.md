# Sandman

## Quick start

Use one mode explicitly:

```text
sandman implement
sandman tdd
sandman review
sandman pr-review
sandman merge
sandman pr-merge
sandman handoff
```

## Modes

- `implement` -> `sandman-implement`
- `tdd` -> `sandman-tdd`
- `review` -> `sandman-review`
- `pr-review` -> `sandman-pr-review`
- `merge` -> `sandman-merge`
- `pr-merge` -> `sandman-pr-merge`
- `handoff` -> `sandman-handoff`

## Use

Load the matching subskill for the requested mode and follow it end to end.
